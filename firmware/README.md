# Firmwares for Xiaomi earbuds

 - Redmi AirDots 3 Pro
 - Redmi Buds 3 Pro
 - Redmi Buds 3
 - Xiaomi Buds 3
 - FlipBuds Pro
 - Redmi Buds 4 Pro
 - Xiaomi Buds 4 Pro
 - Xiaomi Buds 4
 - Redmi Buds 5
 - Redmi Buds 5 AAPE
 - Redmi Buds 5 Pro
 - Redmi Buds 5 Pro Gaming

If you have any firmware files for earbuds not listed here, please, contact me to add the to the list.
